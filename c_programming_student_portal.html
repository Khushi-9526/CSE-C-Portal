Unit ‚Äì I Detailed Notes

1. Introduction to C Language

C is a general-purpose, procedural programming language developed by Dennis Ritchie at AT&T Bell Laboratories. It is widely used in system software, embedded systems, and application development.

2. Features of C

  • Simple and efficient
  • Structured programming language
  • Portable (machine independent)
  • Supports low-level programming using pointers
  • Rich set of operators

3. Structure of a C Program

#include 

int main() {
    printf("Hello World");
    return 0;
}

Explanation: Header files, main function, statements, return statement.

4. Tokens in C

Tokens are the smallest units of a C program.

  • Keywords
  • Identifiers
  • Constants
  • Strings
  • Operators
  • Special symbols

5. Data Types in C

  • Basic: int, char, float, double
  • Derived: array, pointer
  • User-defined: struct, union, enum

6. Variables and Constants

Variables store data values during program execution. Constants store fixed values.

int a = 10; // variable
const float pi = 3.14; // constant

7. Operators in C

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Assignment Operators
  • Increment & Decrement Operators

8. Expressions

An expression is a combination of variables, constants, and operators that produces a value.

int result = a + b * c;

Unit ‚Äì II Theory Notes

1. Control Statements

Control statements alter the normal flow of execution of a program.

  • Selection: if, if-else, else-if ladder, switch
  • Iteration: for, while, do-while
  • Jump: break, continue, goto, return

2. Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations.

3. Strings

A string is an array of characters terminated by a null character ().

Program 3: Swap Two Numbers using Pointers

Aim: To swap two numbers using call by reference.

    int t = *a;

    *b = t;
}

    int x, y;

Program 4: Dynamic Memory Allocation using malloc()

Aim: To allocate memory dynamically using malloc.

#include 
#include 
int main() {
    int n, i;
    int *p;
    scanf("%d", &n);
    p = (int *)malloc(n * sizeof(int));
    if (p == NULL) {
        printf("Memory allocation failed");
        return 0;
    }
    for (i = 0; i < n; i++)
        scanf("%d", &p[i]);
    for (i = 0; i < n; i++)
        printf("%d ", p[i]);
    free(p);
    return 0;
}

Unit ‚Äì V Theory Notes

Structures in C

A structure is a user-defined data type that allows grouping of variables of different data types.

struct student {
    int roll;
    char name[20];
    float marks;
};

Unions in C

A union is similar to a structure, but all members share the same memory location.

union data {
    int i;
    float f;
    char c;
};

Difference between Structure and Union

  • Structure allocates separate memory for each member
  • Union shares the same memory for all members

Enumerations (enum)

Enumeration is a user-defined data type that assigns names to integer constants.

enum days {Mon, Tue, Wed, Thu, Fri};

File Handling in C

File handling allows data to be stored permanently in files.

  • fopen()
  • fclose()
  • fprintf()
  • fscanf()

Preprocessor Directives

Preprocessor directives are instructions processed before compilation.

  • #include
  • #define
  • #ifdef / #ifndef
